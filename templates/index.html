FAQ

How does Note Safe work?

When users create an account, he/she will be given a secret key. Files submitted to the server will be encrypted with a public key. The encrypted file will be stored on the server and the only way to decrypt the file is with the user's secrete key. Note Safe will only ever store encrypted information.


What are Note Safe's limitations?

Because Note Safe is still in its development stage, it only accepts text and it does not support uploads. The accepted size is limited to 300 words per note. Ideally, with further development, Note Safe will support bigger files, uploading and other types of files. There is no limit for how many notes a user can upload.


If I lose my password, can I change my password?

Note Safe does not store any personal information, there is no way to change your password.


If I lose my secret key, can I recover it?

Note Safe never saves user's secret key so once the secret key is lost, it is impossible to recover it. Though this may be an inconvenience, it ensures extra security for users. Please keep your secret key safe and do not lose it.


How long will documents stay on the server?

Documents will stay on the server for an unlimited amount of time, given users log in at least once every few years.


How and where are stuff stored?

Encrypted data is stored in our server. We will never store any decrypted data.


Can I allow access to multiple users?

As long as all users have the username, password, and secret key, multiple users can access the files.


How can I remove files from this site?

Double clicking on a file will bring up a delete dialogue confirming the deletion. Once a file is deleted, the encrypted file will be wiped from our server.
